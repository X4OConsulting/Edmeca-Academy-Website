# Manual GitHub Label Creation Guide

Since the repository is owned by the **X4OConsulting organization**, you need to create the missing labels manually.

## Quick Access

Go to: https://github.com/X4OConsulting/Edmeca-Academy-Website/labels/new

---

## Create These 4 Labels

### 1. needs-work

- **Name:** `needs-work`
- **Description:** `PR requires changes before it can be merged`
- **Color:** `#d93f0b` or hex code: `d93f0b`

Click "Create label"

---

### 2. failing-checks

- **Name:** `failing-checks`
- **Description:** `Build, tests, or checks are failing`
- **Color:** `#b60205` or hex code: `b60205`

Click "Create label"

---

### 3. auto-generated-description

- **Name:** `auto-generated-description`
- **Description:** `PR description was automatically generated by CI/CD`
- **Color:** `#0e8a16` or hex code: `0e8a16`

Click "Create label"

---

### 4. critical-files-changed

- **Name:** `critical-files-changed`
- **Description:** `PR modifies critical infrastructure files requiring extra review`
- **Color:** `#fbca04` or hex code: `fbca04`

Click "Create label"
