---

## Project: EDMECA — Marketing Website + Authenticated Client Portal

Build a modern, production-ready web application for **EDMECA** consisting of two experiences:
1. **Public marketing website** (unauthenticated)
2. **Registered client portal** (authenticated SaaS-style web app with tools)

---

## Brand & Positioning

**Tagline:** "From Framework to Execution to Evidence"

**Core promise (must be clear within 10 seconds):**
EDMECA makes MBA-level business frameworks accessible to entrepreneurs through hands-on delivery + AI-enabled tooling—turning frameworks into action, prototypes into products, and ideas into investor-ready pitches.

**Two audiences:**
- **Entrepreneurs:** Seeking business clarity, faster MVP development, pitch-ready confidence
- **Enterprise Development Programmes / Incubators / Accelerators:** Seeking measurable participant progression, streamlined M&E reporting, scalable interventions

Frontend: Next.js 14+ (App Router), TypeScript, Tailwind CSS, shadcn/ui

## Part 2: Authenticated Client Portal

### Auth & Roles

Implement NextAuth.js with role-based access:

```typescript
enum Role {
  ENTREPRENEUR = 'entrepreneur',
  PARTICIPANT = 'participant', 
  PROGRAMME_MANAGER = 'programme_manager',
  ADMIN = 'admin'
}
```

**Registration flows:**
- Entrepreneur: Self-register → select role → onboarding questions → dashboard
- Participant: Join via invite link/code → linked to Organisation + Cohort
- Programme Manager: Create Organisation → create Cohorts → invite participants

### Database Schema (Prisma)

```prisma
model User {
  id            String   @id @default(cuid())
  email         String   @unique
  name          String?
  passwordHash  String?
  role          Role     @default(ENTREPRENEUR)
  organisationId String?
  organisation  Organisation? @relation(fields: [organisationId], references: [id])
  cohortId      String?
  cohort        Cohort?  @relation(fields: [cohortId], references: [id])
  artifacts     Artifact[]
  progressEntries ProgressEntry[]
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt
}

model Organisation {
  id          String   @id @default(cuid())
  name        String
  users       User[]
  cohorts     Cohort[]
  createdAt   DateTime @default(now())
}

model Cohort {
  id             String   @id @default(cuid())
  name           String
  organisationId String
  organisation   Organisation @relation(fields: [organisationId], references: [id])
  inviteCode     String   @unique @default(cuid())
  users          User[]
  startDate      DateTime?
  endDate        DateTime?
  createdAt      DateTime @default(now())
}

model Artifact {
  id          String   @id @default(cuid())
  userId      String
  user        User     @relation(fields: [userId], references: [id])
  toolType    ToolType
  title       String
  content     Json
  version     Int      @default(1)
  status      ArtifactStatus @default(DRAFT)
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
}

model ProgressEntry {
  id          String   @id @default(cuid())
  userId      String
  user        User     @relation(fields: [userId], references: [id])
  milestone   String
  evidence    String?
  attachmentUrl String?
  completedAt DateTime?
  createdAt   DateTime @default(now())
}

enum ToolType {
  BMC
  SWOT_PESTLE
  VALUE_PROPOSITION
  PITCH_BUILDER
}

enum ArtifactStatus {
  DRAFT
  IN_PROGRESS
  COMPLETE
}
```
